# Redux-E-commerce-Platform

This is an e-commerce platform that uses Redux to manage global state instead of Reactâ€™s Context API. The app allows users to register, browse products by category, view product details, add products to their cart, and checkout using Stripe's API.

## Acceptance Criteria

```md
AS a senior engineer working on an e-commerce platform
I WANT my platform to use Redux to manage global state instead of the Context API
SO THAT my website's state management is taken out of the React ecosystem
```

## Technologies Used

- React
- Redux
- Stripe API
- Express 
- MongoDB

## Installation 

1. Clone the repository to your local machine.
2. Navigate to the root directory of the project in your terminal.
3. Run `npm install` to install dependencies.
4. Run `npm start` to start the development server.
5. Visit `http://localhost:3000` in your web browser to view the app.
